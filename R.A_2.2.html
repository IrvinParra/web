NAT

La NAT convierte varias direcciones IP en una sola, además de que se pueden utilizar para ocultarlas como una interna se puede ocultar una pública o viceversa.

COMANDOS
Router(config-if)#ip nat incidente servere static
Router(config-if)#interface rembertible
Router(config-if)#ip host inside
Router(config-if)#exit
Router(config)#interface number type
Router(config)#ip nat inside

DHCP

Es un protocolo cliente-servidor que proporciona su dirección IP con una máscara de sobredosis además su puerta de enlace previamente predeterminada.

Comandos
R1>enable
R1#config terminal
R1(config)#ip dhcp 192.168.40.1 192.168.40.9
R1(dhcp-config)#ip dhcp excluded-address 192.168.40.10
R1(dhcp-config)#network 192.168.40.0 255.255.255.0
R1(dhcp-config)#default route 192.168.40.1
R1(dhcp-config)#exit
